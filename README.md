# 🏥 Sistema de Gestão Clínica - MedCore

Uma plataforma moderna e intuitiva para gestão de clínicas médicas, focada na eficiência operacional e na experiência do usuário. Desenvolvida com as tecnologias mais recentes do ecossistema React.

## 🚀 Funcionalidades Principais

*   **Dashboard Inteligente:** Visualização rápida de métricas e estatísticas da clínica.
*   **Gestão de Pacientes:** Cadastro completo, histórico e listagem de pacientes.
*   **Gestão de Médicos:** Controle de profissionais por especialidade.
*   **Agendamento de Consultas:** Sistema de marcação e acompanhamento de consultas.
*   **Interface Responsiva:** Design adaptável para diferentes dispositivos.
*   **Context API:** Gerenciamento de estado global para dados da aplicação.

## 🛠️ Tecnologias Utilizadas

*   **[React 19](https://react.dev/):** Biblioteca principal para a interface.
*   **[Vite](https://vitejs.dev/):** Build tool ultra-rápida para desenvolvimento.
*   **[Tailwind CSS](https://tailwindcss.com/):** Framework CSS utilitário para design moderno.
*   **[React Router 7](https://reactrouter.com/):** Navegação entre páginas sem recarregamento.
*   **[Lucide React](https://lucide.dev/):** Conjunto de ícones profissionais.

## 📂 Estrutura do Projeto

```text
src/
├── components/ # Componentes reutilizáveis (Sidebar, Layout, Modais)
├── context/    # Gerenciamento de estado global (AppContext)
├── pages/      # Páginas principais (Dashboard, Pacientes, Médicos)
├── services/   # Simulação de API e Mock Data
└── App.jsx     # Ponto de entrada e rotas
```

## ⚙️ Como Executar o Projeto

### Pré-requisitos
*   [Node.js](https://nodejs.org/) (Versão 18 ou superior)

### Passo a Passo
1. Clone o repositório ou baixe os arquivos.
2. Abra o terminal na pasta do projeto.
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
5. Acesse o endereço indicado no terminal (ex: `http://localhost:5173`).
